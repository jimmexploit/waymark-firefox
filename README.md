# WayMark — Firefox-based Extension

WayMark is a lightweight Firefox extension that lets you **mark positions on web pages and quickly navigate between them**, making it easier to read, revisit, and manage long content.

## ✨ Features

* 📍 **Mark Positions on a Page**
  Drop markers anywhere on a webpage to save important spots for later.

* 🔁 **Quick Navigation**
  Instantly jump between your saved marks without scrolling manually.

* 💾 **Persistent Marks**
  Marks can be saved and reused across sessions (depending on settings), allowing you to return to the same points later.

* ⚡ **Keyboard Shortcuts**
  Use customizable shortcuts to:

  * Add a mark
  * Jump between marks
  * Navigate efficiently

* 🧠 **Improved Reading Workflow**
  Ideal for:

  * Long technical articles
  * Documentation
  * Research papers

---

## 🚀 Use Cases

* Reading long blog posts or writeups
* Navigating documentation while coding
* Studying or reviewing content
* Jumping between important sections quickly

---

## ⚙️ How It Works

1. Open any webpage
2. Add a mark at your current position
3. Use shortcuts or controls to jump between marks
4. Return later and continue where you left off

## 🔐 Privacy

## 💡 Why WayMark?

Scrolling through long pages repeatedly is inefficient.
WayMark gives you **bookmark-like navigation inside a single page**, saving time and improving focus. ([Firefox Add-ons][1])
