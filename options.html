Viewport limit
One mark per type in view
Max one 📌 one-time and one 🔖 saved mark visible on screen at once. Adding another replaces the oldest in view. Scroll down to place more.
Proximity — no overlapping marks
Block marks placed too close together
When you place a new mark, any existing mark of the same type within the radius below is removed automatically.
Proximity radius (px)
How many pixels around a new mark to check for existing same-type marks. Default: 250.
Mark title display
Double-click pin to add title
When enabled, double-clicking a pin on the page opens a title input. When disabled, title can only be added via the sidebar bookmark.
Pin title visibility
When a pin has a title, choose how it appears next to the pin on the page. Sidebar bookmarks always show the title on hover.
One-time mark lifetime
Delete one-time marks when tab closes
One-time marks are already not saved permanently. Enable this to also remove them when you close the tab instead of waiting for a page refresh.

Bookmark folder name

Waymark
Version 1.0.1
Drop pins anywhere on any page. One-time or saved permanently. Navigate back instantly.
Pin anywhere
Right-click or use keyboard shortcuts to drop marks at exact positions on any page.
Drag to move
Pins are draggable. Move them out of your way or drop them on the trash to delete.
Scroll indicator
Bookmark shapes on the right edge show all your marks at a glance. Click to jump.
Search all marks
The Search tab in the popup finds marks across every page you've ever marked.
Export & import
Back up your marks and settings as a single JSON file. Import on any device.
Lightweight
Pure JS/CSS, no dependencies, no network requests. Everything stays in your browser.
